# Aries Restoration

This repository contains a simple prototype for the Aries Restoration company website. The landing page showcases key services and contact information.

## Structure
- `docs/TECHNICAL_TASK.md` – Technical requirements for the site.
- `docs/MVP_PLAN.md` – MVP plan for further development.
- `prototype/` – HTML/CSS prototype with three pages: Landing, About, and Contact.

## Usage
Open `index.html` in a browser. It redirects to the prototype's landing page.
