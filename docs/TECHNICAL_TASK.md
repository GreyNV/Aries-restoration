# Technical Task: Aries Restoration Website

## Overview
Aries Restoration specializes in repairing and restoring properties damaged by disasters. The website should present the company's services and allow potential clients to easily get in touch.

## Pages
1. **Landing Page (`index.html`)**
   - Brief introduction to Aries Restoration and core services.
   - Hero image of restoration work (generated via AI, alt text must reference prompt).
   - Call to action linking to contact page.
2. **About Us Page (`about.html`)**
   - Company history and mission.
   - Team section with optional images (generated via AI) including prompt references in alt text.
3. **Contact Page (`contact.html`)**
   - Contact form (name, email, message) for initial inquiries.
   - Company phone number and email address.

## Non‑Functional Requirements
- Responsive design for mobile and desktop.
- Clean, semantic HTML structure with CSS for styling.
- Image placeholders should be created using AI prompts; the exact prompts must be included in `alt` attributes.

## Deliverables
- HTML prototype containing the three pages.
- Minimal CSS styling.
- README documenting how to run the prototype.
- MVP plan detailing next steps for full development.

## Timeline
The prototype should be created immediately, with iteration and additional features in future phases.
